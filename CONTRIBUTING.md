# Contributing

This is a personal portfolio project, but feedback and suggestions are welcome.

## Reporting Issues

Open a GitHub issue with a clear description of the bug or suggestion.

## Pull Requests

1. Fork the repository and create a feature branch.
2. Follow PSR-12 style — run `./vendor/bin/pint` before committing.
3. Add or update tests for any changed behaviour.
4. Ensure `php artisan test` passes with no failures.
5. Open a pull request with a brief description of what changed and why.
